= Deploying virtual machines on {product-title}

:context: virt-deploying-vms-osd-gcp

toc::[]

[role="_abstract"]
Deploy virtual machines (VMs) on an {product-title} cluster on {gcp-full} by configuring the HyperConverged Operator, creating a {gcp-short} Hyperdisk Balanced storage pool, and creating a custom StorageClass. This configuration optimizes VM performance by leveraging {gcp-short} storage for VM disks.
